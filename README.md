# Alexandria Repuestos

Sitio web de presentación para **Alexandria Repuestos**, un emprendimiento familiar enfocado en la venta de **inyectores y repuestos diésel**.

Actualmente el proyecto está orientado a mostrar el negocio, facilitar el contacto con clientes y destacar productos disponibles, comenzando por el inyector **Bosch 0445120212**.

## Objetivo

Desarrollar una web clara, profesional y funcional que permita:

- presentar la marca
- mostrar productos disponibles
- facilitar consultas por WhatsApp
- transmitir confianza y seriedad
- preparar una base escalable para futuros repuestos y nuevas secciones

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript

## Estado del proyecto

En desarrollo.
